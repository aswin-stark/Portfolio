## 🚀 About This Portfolio

A sleek, neon-themed personal portfolio built with **React + Vite**, featuring an animated starfield background, smooth scroll effects, and a fully responsive design. It showcases my skills, educational background, and provides a direct way to get in touch.

---

## 🖥️ Sections

| Section | Description |
|---|---|
| **Hero** | Introduction with profile picture, live clock, location, and quick-access cards for Resume, LinkedIn, GitHub & Email |
| **About** | Personal summary — who I am, what I do, and my core traits |
| **Skills** | Interactive skill cards grouped by category with a modal pop-up for detailed view |
| **Education** | Academic background — MCA from SRM IST & BCA from Ramakrishna Mission Vivekananda College |
| **Contact** | Message form (EmailJS), social links, and direct contact info |
| **Footer** | Animated gradient border, social icons, and scroll-to-top button |

---

## 🛠️ Tech Stack

**Frontend**
- React 19
- Vite 7
- Tailwind CSS 4
- Framer Motion (header animations)
- Lucide React & React Icons

**Utilities**
- EmailJS — contact form email delivery
- Google Fonts via `@fontsource` (Orbitron, Poppins)
- `gh-pages` — GitHub Pages deployment

---

## ✨ Features

- 🌌 Animated neon starfield background across all sections
- 🖱️ Custom neon cursor effect
- 📜 Scroll-triggered fade-in animations on cards and content
- 💬 Working contact form powered by EmailJS
- 📱 Fully responsive — mobile hamburger menu included
- ⬆️ Scroll-to-top floating button
- 🕐 Live clock and location display in the Hero section
- 🃏 Interactive Skills modal pop-up

## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/aswin-stark/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
